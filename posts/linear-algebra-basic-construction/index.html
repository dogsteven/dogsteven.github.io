Vector spaces, subspaces, linear independence, bases 🔗

Definition 1. Let $\mathbb{K}$ be a field. A vector space over $\mathbb{K}$ is a non-empty set $V$ together with two operations $+ : V\times V \to V$ and $\cdot : \mathbb{K}\times V\to V$, called the vectors addition and the scalar multiplication, respectively, which satisfies the following axioms

  1. $(V, +)$ is an Abelian group, we denote the its identity element by $\mathbf{0}$.
  2. $1\cdot \mathbf{v} = \mathbf{v}$ for every $\mathbf{v}\in V$.
  3. $(ab)\cdot \mathbf{v} = a\cdot (b\cdot \mathbf{v})$ for every $a, b\in \mathbb{K}$ and $\mathbf{v}\in V$.
  4. $(a + b)\cdot \mathbf{v} = a\cdot \mathbf{v} + b\cdot \mathbf{v}$ for every $a, b\in \mathbb{K}$ and $\mathbf{v}\in V$.
  5. $a\cdot (\mathbf{v} + \mathbf{w}) = a\cdot \mathbf{v} + a\cdot \mathbf{w}$ for every $a\in \mathbb{K}$ and $\mathbf{v}, \mathbf{w}\in V$.

The elements of a vector space are called vectors, and the vector $\mathbf{0}$ is called zero vector. The elements of the field $\mathbb{K}$ are called scalars.

Proposition 1. $0\cdot \mathbf{v} = \mathbf{0}$ for every $\mathbf{v}\in V$.

Proposition 2. For every $a\in \mathbb{K}$ and $\mathbf{v}\in V$, we have $a\cdot \mathbf{v} = \mathbf{0}$ implies that $a = 0$ or $\mathbf{v} = \mathbf{0}$.

Definition 2. Let $V$ be a vector space over a field $\mathbb{K}$. A non-empty subset $U$ of $V$ is called a vector subspace of $V$, or a subspace of $V$ for short, if $U$ is closed under the vectors additiion and the scalar multiplication, i.e.

  1. For every $\mathbf{v}, \mathbf{w}\in U$, the vector $\mathbf{v} + \mathbf{w}$ belongs to $U$.
  2. For every scalar $a\in \mathbb{K}$ and $\mathbf{v}\in U$, the vector $a\cdot \mathbf{v}$ belongs to $U$.

We can see that, a subspace of a vector space is also a vector space over the same field.

Proposition 3. Let $V$ be a subspace and let $\left\{U_i\right\}_{i\in \mathcal{I}}$ be a collection of subspaces of $V$. Then the intersection $\bigcap\limits_{i\in \mathcal{I}} U_i$ is also a subspace of $V$.

Definition 3. Let $A$ be a subset of a vector space $V$. The smallest subspace of $V$ containing $A$ is called the subspace generated by $A$. We denote this space by $\left<A\right>$. We say that $A$ generates $W$ if $\left<A\right> = W$ for some subspace $W$ of $V$. Also, if $\mathbf{v}\in \left<A\right>$, we say that $A$ spans $\mathbf{v}$.

Definition 4. Let $A$ be a subset of a vector space $V$ over a field $\mathbb{K}$. A vector $\mathbf{v}$ in $V$ is said to be a linear combination of $A$ if there is a finite subset $\{\mathbf{v}_1, \mathbf{v}_2, …, \mathbf{v}_n\}$ of $A$ and scalars $a_1, a_2, …, a_n$ such that $\mathbf{v} = a_1\cdot \mathbf{v}_1 + a_2\cdot \mathbf{v}_2 + … + a_n\cdot \mathbf{v}_n$. As a convention, the zero vector $\mathbf{0}$ is the only linear combination of the empty set.

Theorem 1. Let $A$ be a non-empty subset of a vector space $V$. Then the subspace generated by $A$ is exactly the same as the set of all linear combinations of $A$.

Definition 5. Let $V$ be a vector space over a field $\mathbb{K}$. A subset $A$ of $V$ is said to be linear independent if for every finite subset $\left\{\mathbf{v}_1, \mathbf{v}_2, …, \mathbf{v}_n\right\}$ of $A$, for every $a_1, a_2, …, a_n$ in the field $\mathbb{K}$, then $$a_1\cdot\mathbf{v}_1 + a_2\cdot\mathbf{v}_2 + … + a_n\cdot\mathbf{v}_n = \mathbf{0}$$

implies $a_1 = a_2 = … = a_n = 0$. Otherwise, we say $A$ is linear dependent. A subset $A$ is called a basis for $V$ of $A$ is linear independent and $A$ generates $V$.

Proposition 4. Let $A$ be a subset of a vector space $V$. Suppose that $A$ is linear independent and there is a vector $\mathbf{v}\in V\setminus \left<A\right>$. Then $\{\mathbf{v}\}\cup A$ is linear independent.

Theorem 2. Let $A$ be a subset of a vector space $V$ over a field $\mathbb{K}$. Suppose $A$ generates $V$, then $A$ contains a basis for $V$.

Proof. Let $\mathcal{L}$ be the set of all linear independent subsets of $A$. Since $\varnothing\in \mathcal{L}$, then $\mathcal{L}$ is non-empty.

Given a non-empty chain $\{C_i\}_{i\in \mathcal{I}}$ in $\mathcal{L}$, let $C = \bigcup\limits_{i \in \mathcal{I}} C_i$, we show that $C$ is also linear independent. Given a finite subset $\{\mathbf{v}_1, \mathbf{v}_2, …, \mathbf{v}_n\}$ of $C$, suppose that $$a_1\cdot\mathbf{v}_1 + a_2\cdot\mathbf{v}_2 + … + a_n\cdot\mathbf{v}_n = \mathbf{0}$$ for some scalars $a_1, a_2, …, a_n$. For each $j\in \{1, 2, …, n\}$, there is an index $i_j\in \mathcal{I}$ such that the vector $\mathbf{v}_j$ belongs to $C_{i_j}$. Since $\{C_i\}_{i\in \mathcal{I}}$ is a chain, there is an index $k\in \mathcal{I}$ such that all the vectors $\mathbf{v}_1, \mathbf{v}_2, …, \mathbf{v}_n$ is contained in $C_k$. We have $a_1 = a_2 = … = a_n = 0$ as $C_k$ is linear independent. Since $\mathbf{v}_1, \mathbf{v}_2, …, \mathbf{v}_n$ are arbitrarily chosen, it follows that $C$ is linear independent. Obviously, $C$ is a subset of $A$, so $C$ lies in $\mathcal{L}$.

By Zorn’s lemma, $\mathcal{L}$ has a maximal element, which is a maximal linear independent subset of $A$, denote this element by $C$. We show that $C$ generates $V$.

Suppose that $C$ doesn’t generate $V$. There is a vector $\mathbf{v}\in A$ such that $\mathbf{v}\not\in \left<C\right>$. Let’s consider the set $D = \{\mathbf{v}\}\cup C$. This set is a subset of $A$, also this set is linear independent, so $D$ belongs to $\mathcal{L}$, which contradicts to the maximality of $C$.

Therefore, $C$ generates $V$, so $C$ is a basis for $V$.

Theorem 3. Every vector space has a basis.

Theorem 4. Let $A$ be a linear independent subset of a vector space $V$ over a field $\mathbb{K}$. Then we can extend $A$ into a basis for $V$.

Proof. Let $\mathcal{L}$ be the set of all subsets $B$ of $V$ such that $B$ is linear independent and $B$ contains $A$.

Obviously, $A$ is an element of $\mathcal{L}$. Given a chain $\{C_i\}_{i\in \mathcal{I}}$ in $\mathcal{L}$, let $C = \bigcup\limits_{i \in \mathcal{I}} C_i$. From the proof of Theorem 2, $C$ is linear independent. Also, $C$ contains $A$, so $C$ is an element of $\mathcal{L}$.

By Zorn’s lemma, $\mathcal{L}$ has a maximal element, denoted $C$. By a similar argument as Theorem 2, one can show that $C$ is a basis for $V$.

Definition 6. A vector space $V$ is called a finite dimensional vector space if there is a finite subset $A$ of $V$ such that $A$ generates $V$, equivalently, $V$ has a finite basis. Otherwise, we call $V$ an infinite dimensional vector space.

Proposition 5. Let $V$ be a finite dimensional vector space with basis $\{\alpha_1, \alpha_2, …, \alpha_n\}$. Suppose that $\{\mathbf{v}_1, \mathbf{v}_2, …, \mathbf{v}_m\}$ is a linear independent subset of $V$. Then $m\leqslant n$.

Proof. Suppose that $m > n$.

We can write $\mathbf{v}_1 = a_1\cdot \alpha_1 + a_2\cdot \alpha_2 + … a_n\cdot \alpha_n$, since $\mathbf{v}_1\ne \mathbf{0}$, there is an index $i$ such that $a_i \ne 0$. Without loss of general, assume that $a_1 \ne 0$, then $$\alpha_1 = \dfrac{1}{a_1}\cdot \mathbf{v}_1 - \dfrac{a_2}{a_1}\cdot \alpha_2 - … - \dfrac{a_n}{a_1}\cdot \alpha_n$$ so $\alpha_1$ is a combination of $\{\mathbf{v}_1, \alpha_2, …, \alpha_n\}$. Therefore $$\{\alpha_1, \alpha_2, …, \alpha_n\}\subseteq \left<\mathbf{v}_1, \alpha_2, …, \alpha_n\right>$$ so $\{\mathbf{v}_1, \alpha_2, …, \alpha_n\}$ generates $V$.

Now, we can write $\mathbf{v}_2 = a_1\cdot \mathbf{v}_1 + a_2\cdot \alpha_2 + … + a_n\cdot \alpha_n$. If $a_2 = a_3 = … = a_n = 0$, then $\mathbf{v}_2 = a_1\cdot \mathbf{v}_1$, which contradicts to the linear independence of $\{\mathbf{v}_i\}_{i = 1}^{m}$. Without loss of general, assume that $a_2 \ne 0$, by a similar argument, $\{\mathbf{v}_1, \mathbf{v}_2, \alpha_3, …, \alpha_n\}$ generates $V$. And so on. Since $m > n$, after a finite number of steps, we have $\{\mathbf{v}_1, \mathbf{v}_2, …, \mathbf{v}_n\}$ generates $A$, so $\mathbf{v}_m$ is a linear combination of this set, which contradicts to the linear independence of $\{\mathbf{v}_i\}_{i = 1}^{m}$.

Hence the assumption is wrong. In another words, $m\leqslant n$.

Theorem 5. Let $V$ be a vector space. Then any two bases of $V$ have the same cardinality.

Proof. If $V$ is finite dimensional, it follows from Proposition 5 that any two bases of $V$ have the same number of elements. Suppose that $V$ is infinite dimensional.

Let $\mathcal{A}$ and $\mathcal{B}$ be two bases of $V$. For each $\alpha\in \mathcal{A}$, there is a finite subset $\mathcal{F}_{\alpha}$ of $\mathcal{B}$ such that $\alpha$ is a combination of $\mathcal{F}_{\alpha}$. The set $\mathcal{F} = \bigcup\limits_{\alpha\in \mathcal{A}} \mathcal{F}_{\alpha}$ spans $\alpha$ for every $\alpha\in \mathcal{A}$, so $\mathcal{F}$ generates $V$. Therefore $$\mathcal{B} = \bigcup\limits_{\alpha\in \mathcal{A}} \mathcal{F}_{\alpha}$$ We have $$\left|\mathcal{B}\right| = \left|\bigcup\limits_{\alpha\in \mathcal{A}} \mathcal{F}_{\alpha}\right| \leqslant \aleph_0\cdot \left|\mathcal{A}\right| = \left|\mathcal{A}\right|$$ The last equality holds since $\mathcal{A}$ is infinite. Similarly, we have $\left|\mathcal{A}\right| \leqslant \left|\mathcal{B}\right|$.

By Schroder-Bernstein theorem, we have $\mathcal{A}$ and $\mathcal{B}$ have the same cardinality.

Definition 7. Let $V$ be a finite dimensional vector space, the dimension of $V$ is defined to be the number of elements of a basis of $V$, we denote it by $\dim V$.

Theorem 6. Let $V$ be a finite dimensional vector space and let $W$ be its subspace, then $\dim W\leqslant \dim V$. Furthermore, if $\dim W = \dim V$, then $W = V$.

Linear maps, quotient subspaces 🔗

Definition 8. Let $V$ and $W$ be vector spaces over the same field $\mathbb{K}$. A linear map (or a linear homomorphism) is a map $\varphi: V\to W$ satisfying the following condition: For every $\mathbf{v}, \mathbf{u}\in V$ and for every $a, b\in \mathbb{K}$, we have $\varphi(a\cdot \mathbf{v} + b\cdot \mathbf{u}) = a\cdot \varphi(\mathbf{v}) + b\cdot \varphi(\mathbf{u})$.

A linear isomorphism is a bijective linear homomorphism. If there is a linear isomorphism from $V$ into $W$, we say that $V$ is isomorphic to $W$, and we write $V\cong W$.

Proposition 6. Composite of two linear homomorphisms is also a linear homomorphism.

Proposition 7. If $V$ is a vector space and $W$ is a subspace of $V$, then the inclusion $j: W\to V$ is a linear homomorphism.

Corollary 1. We can construct new linear homomorphism by extending the codomain, or restricting the domain of an existing one. Also, we can restrict the codomain of a linear homomorphism to get a new one as long as the new codomain contains the image of the old one.

Proposition 8. Inverse of a linear isomorphism is also a linear isomorphism.

Definition 9. Let $V$ and $W$ be vector spaces over the same field $\mathbb{K}$. The set of all linear homomorphisms from $V$ into $W$ is denoted by $\mathcal{L}(V, W)$. Furthermore, $\mathcal{L}(V, W)$ is a vector space over $\mathbb{K}$ with the following operations for $\varphi, \psi\in \mathcal{L}(V, W)$

  1. $(\varphi + \psi)(\mathbf{v}) = \varphi(\mathbf{v}) + \psi(\mathbf{v})$ for every $\mathbf{v}\in V$.
  2. $(a\cdot \varphi)(\mathbf{v}) = a\cdot \varphi(\mathbf{v})$ for every $\mathbf{v}\in V$.

The zero vector of this space is the zero map, which carries every vector into $\mathbf{0}$.

Definition 10. Let $V$ be a vector space and $W$ be a subspace of $V$. The quotient group $V / W$ together with the scalar multiplication $a\left[\mathbf{v}\right] = \left[a\cdot \mathbf{v}\right]$ forms a vector space called the quotient subspace $V / W$.

Proposition 9. If $V$ is a finite dimensional vector space, and $W$ is a subspace of $V$, then $\dim V / W = \dim V - \dim W$.

Definition 11. Let $V$ and $W$ be vector spaces over the same field, let $\varphi: V\to W$ be a linear homomorphism. The kernel of $\varphi$ is defined to be the set $\left\{\mathbf{v}\in V : \varphi(\mathbf{v}) = \mathbf{0}\right\}$. We denote ther kernel of a homomorphism $\varphi$ by $\ker \varphi$.

Proposition 10. Image of a subspace under a linear homomorphism is a subspace. In particular, the image of a linear homomorphism is a subspace.

Proposition 11. Inverse image of a subspace under a linear homomorphism is a subspace. In particular, the kernel of a linear homomorphism is a subspace.

Proposition 12. Inverse image of a linear independent subset under a linear homomorphism is a linear independent. In particular, image of a linear independent subset under an injective linear homomorphism is also linear independent.

Proposition 13. A homomorphism is injective if and only if its kernel is $\{\mathbf{0}\}$.

Proposition 14. Let $\varphi: V\to W$ be a linear homomorphism from a vector space $V$ to a vector space $W$. For every subset $A$ of $V$, we have $$\left<\varphi(A)\right> = \varphi\left(\left<A\right>\right)$$

Theorem 7. Let $V$ and $W$ be vector spaces over the same field, let $\varphi: V\to W$ be a linear homomorphism. Then there is an injective linear homomorphism $\overline{\varphi}: V / \ker \varphi \to W$ such that $\overline{\varphi}(\left[\mathbf{v}\right]) = \varphi(\mathbf{v})$ for every $\mathbf{v}\in V$. Therefore, $$V / \ker \varphi \cong \varphi(V)$$

Corollary 2. Let $V$ and $W$ be finite dimensional vector spaces over the same field, let $\varphi: V\to W$ be a linear homomorphism, then $$\dim V = \dim \ker \varphi + \dim \varphi(V)$$

Corollary 3. Let $V$ and $W$ be finite dimensional vector spaces and let $\varphi: V\to W$ be a linear homomorphism. Then $$\dim \varphi(V) \leqslant \dim V$$

Proposition 15. Suppose that $V$ is a finite dimensional vector space and $\varphi: V\to V$ is a linear homomorphism. Then the following statements are equivalent:

  1. $\varphi$ is injective.
  2. $\varphi$ is surjective.
  3. $\varphi$ is an isomorphism.

Theorem 8. Let $V$ and $W$ be vector spaces over the same field, let $\mathcal{A}$ be a basis for $V$. Then we can extend any map $\varphi: \mathcal{A}\to W$ to a linear homomorphism from $V$ into $W$. Furthermore, the extension is unique.

Proof. Let $\psi: V\to W$ be defined as follows: Given $\mathbf{v}\in V$, we can write $$\mathcal{v} = \sum\limits_{\alpha\in \mathcal{I}} a_{\alpha}\cdot \alpha$$ for some finite subset $\mathcal{I}$ of $\mathcal{A}$ and some scalars $\{a_{\alpha}\}_{\alpha\in \mathcal{I}}$. Define $$\psi(\mathcal{v}) = \sum\limits_{\alpha\in \mathcal{I}} a_{\alpha}\cdot \varphi(\alpha)$$ First, we show that this definition is well-defined. Given $\mathbf{v} \in V$, suppose that we can write $\mathcal{v}$ as a combination of $\mathcal{A}$ in two ways: $$\mathbf{v} = \sum\limits_{\alpha\in \mathcal{I}} a_{\alpha}\cdot \alpha = \sum\limits_{\alpha\in \mathcal{J}} b_{\alpha}\cdot \alpha$$ Then $$\sum\limits_{\alpha\in \mathcal{I}\setminus \mathcal{J}} a_{\alpha}\cdot \alpha - \sum\limits_{\alpha\in \mathcal{J}\setminus \mathcal{I}} b_{\alpha}\cdot \alpha + \sum\limits_{\alpha\in \mathcal{I}\cap \mathcal{J}} (a_{\alpha} - b_{\alpha})\cdot \alpha = \mathbf{0}$$ By the linear independence of $\mathcal{A}$, $a_{\alpha} = 0$ for every $\alpha\in \mathcal{I}\setminus \mathcal{J}$, $b_{\alpha} = 0$ for every $\alpha \in \mathcal{J}\setminus \mathcal{I}$, and $a_{\alpha} = b_{\alpha}$ for every $\alpha\in \mathcal{I}\cap \mathcal{J}$. Hence $$\sum\limits_{\alpha\in \mathcal{I}} a_{\alpha}\cdot \varphi(\alpha) = \sum\limits_{\alpha\in \mathcal{I}\cap \mathcal{J}} a_{\alpha}\cdot \varphi(\alpha) = \sum\limits_{\alpha\in \mathcal{I}\cap \mathcal{J}} b_{\alpha}\cdot \varphi(\alpha) = \sum\limits_{\alpha\in \mathcal{J}} b_{\alpha}\cdot \varphi(\alpha)$$ It follows that the definition of $\psi$ is well-defined. Now, we show that $\psi$ is a linear homomorphism.

Given vectors $\mathbf{v}, \mathbf{u}\in V$, suppose that $$\mathbf{v} = \sum\limits_{\alpha\in \mathcal{I}} a_{\alpha}\cdot \alpha,\;\; \mathbf{u} = \sum\limits_{\alpha\in \mathcal{J}} b_{\alpha}\cdot \alpha$$ then we have $$\mathbf{v} + \mathbf{u} = \sum\limits_{\alpha\in \mathcal{I}\setminus \mathcal{J}} a_{\alpha}\cdot \alpha + \sum\limits_{\alpha\in \mathcal{I}\cap \mathcal{J}} (a_{\alpha} + b_{\alpha})\cdot \alpha + \sum\limits_{\alpha\in \mathcal{J}\setminus \mathcal{I}} b_{\alpha}\cdot \alpha$$ so $$ \begin{alignat*}{2} \psi(\mathbf{v} + \mathbf{u}) & = \sum\limits_{\alpha\in \mathcal{I}\setminus \mathcal{J}} a_{\alpha}\cdot \varphi(\alpha) + \sum\limits_{\alpha\in \mathcal{I}\cap \mathcal{J}} (a_{\alpha} + b_{\alpha})\cdot \varphi(\alpha) + \sum\limits_{\alpha\in \mathcal{J}\setminus \mathcal{I}} b_{\alpha}\cdot \varphi(\alpha) \\ & = \sum\limits_{\alpha\in \mathcal{I}} a_{\alpha}\cdot \varphi(\alpha) + \sum\limits_{\alpha\in \mathcal{J}} b_{\alpha}\cdot \varphi(\alpha) = \psi(\mathbf{v}) + \psi(\mathbf{u}) \end{alignat*} $$ In another hand, given a scalar $a$, $$a\cdot \mathbf{v} = a\cdot \sum\limits_{\alpha\in \mathcal{I}} a_{\alpha}\cdot \alpha = \sum\limits_{\alpha\in \mathcal{I}} (aa_{\alpha})\cdot \alpha$$ so $$ \psi(a\cdot \mathbf{v})= \sum\limits_{\alpha\in \mathcal{I}} (aa_{\alpha})\cdot \varphi(\alpha) = a\cdot \sum\limits_{\alpha\in \mathcal{I}} a_{\alpha}\cdot \varphi(\alpha) = a\cdot \psi(\mathbf{v}) $$ Therefore $\psi$ is a linear homomorphism, furthermore, $\psi$ is an extension of $\varphi$ from $\mathcal{A}$ to $V$.

Finally, we shwo that the extension is unique. Suppose that $\phi: V\to W$ be a linear map such that $\phi(\alpha) = \mathbf{0}$ for every $\alpha\in \mathcal{A}$. Given $\mathbf{v}\in V$, we can write $$\mathbf{v} = \sum\limits_{\alpha\in \mathcal{I}} a_{\alpha}\cdot \alpha$$ for some finite subset $\mathcal{I}$ of $\mathcal{A}$ and some scalars $\{a_{\alpha}\}_{\alpha\in \mathcal{I}}$, then $$\phi(\mathbf{v}) = \sum\limits_{\alpha\in \mathcal{I}} a_{\alpha} \cdot \varphi(\alpha) = \mathbf{0}$$

Thus, $\phi$ is the zero map. Suppose that $\psi^{\star}: V\to W$ is a linear homomorphism which is an extension of $\varphi$, then for every $\alpha\in \mathcal{A}$, we have $(\psi - \psi^{\star})(\alpha) = \mathbf{0}$, hence $\psi - \psi^{\star}$ is the zero map. Therefore $\psi = \psi^{\star}$, which proves the uniqueness of $\psi$.

Theorem 9. Let $V$ and $W$ be vector spaces over the same field. Let $\mathcal{A}$ and $\mathcal{B}$ be bases for $V$ and $W$, respectively. Then $V\cong W$ if and only if $\left|\mathcal{A}\right| = \left|\mathcal{B}\right|$.

Direct products, direct sums 🔗

Definition 12. Let $\left\{V_{i}\right\}_{i\in \mathcal{I}}$ be a collection of vector spaces over the same field. The Cartesian product of this collection together with two following operations

  1. $\left(\mathbf{v}_i\right)_{i \in \mathcal{I}} + \left(\mathbf{u}_i\right)_{i \in \mathcal{I}} = \left(\mathbf{v}_i + \mathbf{u}_i\right)_{i \in \mathcal{I}}$, and
  2. $a\cdot \left(\mathbf{v}_i\right)_{i \in \mathcal{I}} = \left(a\cdot \mathbf{v}_i\right)_{i \in \mathcal{I}}$

is a vector space, called the direct product of the collection $\left\{V_{i}\right\}_{i\in \mathcal{I}}$, denoted by $$\prod\limits_{i\in \mathcal{I}} V_{i}$$

Definition 13. For every $j\in \mathcal{I}$, the map $$ \begin{alignat*}{2} \pi_j: \prod\limits_{i\in \mathcal{I}} V_{i} & \to V_{j} \\ (\mathbf{v}_{i})_{i\in \mathcal{I}} & \mapsto \mathbf{v}_{j} \end{alignat*} $$ is a linear homomorphism, called the $j$-th coordinate projection.

Theorem 10. Let $\left\{V_{i}\right\}_{i\in \mathcal{I}}$ be a collection of vector spaces over the same field. Then for every vector space $V$, for every collection of linear homomorphisms $\left\{\varphi_i: V\to V_{i}\right\}_{i\in \mathcal{I}}$, there is a unique linear homomorphism $\varphi: V\to \prod\limits_{i\in \mathcal{I}} V_{i}$ such that $$\pi_i\circ \varphi = \varphi_i$$ for every $i\in \mathcal{I}$. More explicitly, $$\varphi(\mathbf{v}) = \left(\varphi_i(\mathbf{v})\right)_{i\in \mathcal{I}}$$ for every $\mathbf{v}\in V$.

Definition 14. Let $\left\{V_{i}\right\}_{i\in \mathcal{I}}$ be a collection of vector spaces over the same field. The subcollection of the Cartesian product of $\left\{V_{i}\right\}_{i\in \mathcal{I}}$ consisting of all tuples with finitely many nonzero vectors is a subspace of the product space of $\left\{V_{i}\right\}_{i\in \mathcal{I}}$. This space is denoted by $$\bigoplus\limits_{i\in \mathcal{I}} V_{i}$$

Definition 15. For every $j\in \mathcal{I}$, the map $$ \begin{alignat*}{2} \mathbf{i}_j: V_j & \to \bigoplus\limits_{i\in \mathcal{I}} V_{i} \\ \mathbf{v} & \mapsto (…, \mathbf{0}, \mathbf{v}, \mathbf{0}, …) \end{alignat*} $$ carrying each $\mathbf{v}\in V_j$ into a tuple having value $\mathbf{v}$ at index $j$ and zero otherwise, is a linear homomorphism. This linear homomorphism is called the $j$-th coordinate injection. For each $\mathbf{v}\in V_i$ and $\mathbf{u}\in V_j$ for some $i, j\in \mathcal{I}$, we define $$\mathbf{v} \oplus \mathbf{u} = \mathbf{i}_i(\mathbf{v}) + \mathbf{i}_j(\mathbf{u})$$ We can extend this definition to sums of $n$-terms $$\mathbf{v}_1 \oplus \mathbf{v}_2 \oplus … \oplus \mathbf{v}_n$$ By definition, every element of the direct product can be represented as a sum of a finite number of terms.

Theorem 11. Let $\left\{V_{i}\right\}_{i\in \mathcal{I}}$ be a collection of vector spaces over the same field. Then for every vector space $V$, for every collection of linear homomorphisms $\left\{\varphi_i: V_i\to V\right\}_{i\in \mathcal{I}}$, there is a unique linear homomorphism $\varphi: \bigoplus\limits_{i\in \mathcal{I}} V_{i}\to V$ such that $$\varphi_i = \varphi\circ \mathbf{i}_i$$ for every $i\in \mathcal{I}$. More explicitly, $$\varphi\left(\mathbf{v}_1 \oplus \mathbf{v}_2 \oplus … \oplus \mathbf{v}_n\right) = \sum\limits_{k = 1}^{n} \varphi_{i_k}(\mathbf{v}_k)$$ where $\mathbf{v}_k \in V_{i_k}$ for every $k\in \{1, 2, …, n\}$

Corollary 4. Let $\{V_i\}_{i\in \mathcal{I}}$ be a collection of vector spaces, let $W$ be a vector space. Then $$\mathcal{L}\left(\bigoplus\limits_{i\in \mathcal{I}} V_i, W\right) \cong \prod\limits_{i\in \mathcal{I}} \mathcal{L}(V_i, W)$$

Proposition 16. Let $V$ be a vector space, let $\left\{V_{i}\right\}_{i\in \mathcal{I}}$ be a collection of subspaces of $V$. By the preceeding theorem, there is a unique linear homomorphism $$\varphi: \bigoplus\limits_{i\in \mathcal{I}} V_{i}\to V$$ such that for every $i\in \mathcal{I}$, we have $\varphi\circ \mathbf{i}_i = \mathbf{j}_i$ where $\mathbf{j}_i: V_i\to V$ is the inclusion map. The image of this linear homomorphism is a subspace of $V$, which we denote by $\sum\limits_{i\in \mathcal{I}} V_i$. Then $$\sum\limits_{i\in \mathcal{I}} V_i = \left<\bigcup\limits_{i\in \mathcal{I}} V_i\right>$$

Theorem 12. Let $V$ be a vector space, let $\left\{V_{i}\right\}_{i\in \mathcal{I}}$ be a collection of subspaces of $V$. Then $$\sum\limits_{i\in \mathcal{I}} V_i \cong \bigoplus\limits_{i\in \mathcal{I}} V_i$$ if and only if for every $i\in \mathcal{I}$, we have $$V_i \cap \left(\sum\limits_{j\in \mathcal{I}\setminus \{i\}} V_j\right) = \{\mathbf{0}\}$$

Theorem 13. Let $V$ be a vector space, let $\left\{V_{i}\right\}_{i = 1}^{n}$ be a finite collection of subspaces of $V$. Suppose that $$V \cong \bigoplus\limits_{i = 1}^{n} V_i$$ Then every vector of $V$ can be written uniquely as the form $$\mathbf{v}_1 + \mathbf{v}_2 + … + \mathbf{v}_n$$ where $\mathbf{v}_i\in V_i$ for every $i\in \{1, 2, …, n\}$.

Theorem 14. Every vector space of dimension $n\in \mathbb{Z}^{+}$ is isomorphic to $\mathbb{K}^n$.

Theorem 15. Let $V$ be a vector space and let $W$ be a subspace of $V$. There is a subspace $U$ of $V$ such that $V \cong W\oplus U$. These such subspaces $U$ are non-unique and are called algebraic complements of $W$ in $V$.

Proof. Let $\mathcal{B}$ be a basis for $W$, then we can extend $\mathcal{B}$ into a basis $\mathcal{A}$ for $V$, let $\mathcal{C} = \mathcal{A}\setminus \mathcal{B}$, let $U$ is the subspace generated by $\mathcal{C}$. We show that $V \cong W\oplus U$.

Given $\mathbf{v} \in W\cap U$, we can write $$\mathbf{v} = \sum\limits_{\beta\in \mathcal{I}} b_{\beta}\cdot \beta = \sum\limits_{\gamma\in \mathcal{J}} c_{\gamma}\cdot \gamma$$ for some finite subsets $\mathcal{I}$ and $\mathcal{J}$ of $\mathcal{B}$ and $\mathcal{C}$, respectively. Then $$\sum\limits_{\beta\in \mathcal{I}} b_{\beta}\cdot \beta - \sum\limits_{\gamma\in \mathcal{J}} c_{\gamma}\cdot \gamma = \mathbf{0}$$ By the linear independence of $\mathcal{A}$, we have $b_{\beta} = 0$ for every $\beta\in \mathcal{I}$, so $\mathbf{v} = \mathbf{0}$.

Hence $W\cap U = \{\mathbf{0}\}$, therefore $W\oplus U \cong W + U = \left<W\cup U\right> = \left<\mathcal{B}\cup \mathcal{C}\right> = \left<\mathcal{A}\right> = V$.

Definition 16. Let $\{V_i\}_{i\in \mathcal{I}}$ be a collection of vector spaces. A vector space $V$ together with a collection $\{\mathbf{i}_i: V_i\to V\}_{i\in \mathcal{I}}$ of linear homomorphisms is called a direct sum of $\{V_i\}_{i\in \mathcal{I}}$ if for every collection $\{\varphi_i: V_i\to Z\}_{i\in \mathcal{I}}$ of linear homomorphisms, there is a unique linear homomorphism $\varphi: V\to Z$ such that $$\varphi_i = \varphi\circ \mathbf{i}_i$$ for every $i\in \mathcal{I}$.

Theorem 16. The space $V = \bigoplus\limits_{i\in \mathcal{I}} V_{i}$ together with the collection $\left\{\mathbf{i}_i : V_i\to V\right\}_{i\in \mathcal{I}}$ is a direct sum of $\{V_i\}_{i\in \mathcal{I}}$.

Theorem 17. Direct sum is unique up to isomorphism.

Proof. Let $\left(V, \{\mathbf{i}_i^V: V_i\to V\}_{i\in \mathcal{I}}\right)$ and $\left(Z, \{\mathbf{i}_i^Z: V_i\to Z\}_{i\in \mathcal{I}}\right)$ be two direct sums of $\{V_i\}_{i\in \mathcal{I}}$.

Consider the collection $\{\mathbf{i}_i^V: V_i\to V\}_{i\in \mathcal{I}}$, there is a unique linear homomorphism $\varphi: Z\to V$ such that $\varphi\circ \mathbf{i}_i^Z = \mathbf{i}_i^V$ for every $i\in \mathcal{I}$.

Consider the collection $\{\mathbf{i}_i^Z: V_i\to Z\}_{i\in \mathcal{I}}$, there is a unique linear homomorphism $\psi: V\to Z$ such that $\psi\circ \mathbf{i}_i^V = \mathbf{i}_i^Z$ for every $i\in \mathcal{I}$.

Therefore, for every $i\in \mathcal{I}$ $$\mathbf{i}_i^V = \varphi\circ \mathbf{i}_i^Z = \varphi\circ \psi \circ \mathbf{i}_i^V\text{ and }\mathbf{i}_i^Z = \psi\circ \mathbf{i}_i^V = \psi\circ \varphi \circ \mathbf{i}_i^Z$$

Consider the collection $\{\mathbf{i}_i^V: V_i\to V\}_{i\in \mathcal{I}}$, there is a unique linear homomorphism $j: V\to V$ such that $j = j\circ \mathbf{i}_i^V$ for every $i\in \mathcal{I}$. Since $\mathbf{id}_V \circ \mathbf{i}_i^V = \mathbf{i}_i^V$ for every $i\in \mathcal{I}$, by the uniqueness, $$\varphi\circ \psi = \mathbf{id}_V$$ Similarly, $\psi\circ \varphi = \mathbf{id}_Z$. So $\varphi$ is a linear isomorphism.
