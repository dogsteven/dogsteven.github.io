Group Theory: Composition Series

Jul 9, 2022 ┬Ę 2158 words ┬Ę 11 minute read

Theorem 1. Let $G$ be a finite group, let $p$ be a prime divisor of $|G|$. Then $G$ has an element of order $p$.

Proof. Consider the following set $$\mathcal{S} = \left\{(a_1, a_2, …, a_p)\in G^n : a_1a_2…a_p = 1\right\}$$ Let $H = \left<\sigma\right>\leqslant S_{G^p}$ where $\sigma$ is a permutation on $G^n$ defined by $$\sigma(a_1, a_2, …, a_p) = (a_2, a_3, …, a_p, a_1)$$

First, we observe that $$\sigma^{k}(a_1, a_2, …, a_p) = (a_{k + 1\text{ mod } p}, a_{k + 2\text{ mod } p}, …, a_{k + p\text{ mod } p})$$ for every $a_1, a_2, …, a_p\in G$ and $k\in \mathbb{Z}^{+}$. So $$\sigma^p(a_1, a_2, …, a_p) = (a_1, a_2, …, a_p)$$ so $H\cong \mathbb{Z}_p$. Next, $\sigma$ doesn’t change the product of $p$-tuples, so let $H$ acts on $\mathcal{S}$ in a obvious way.

Let $H\mathbf{a}$ be the orbit of an element $\mathbf{a}\in \mathcal{S}$, then $$\left|H\mathbf{a}\right| = [H : H_{\mathbf{a}}] \mid p$$ Hence each orbit of this action has size $1$ or $p$. On the other hand, $$p \mid |G|^{n - 1} = |\mathcal{S}| = \sum\limits_{\mathcal{O}\in \mathcal{S} / H} |\mathcal{O}|$$ and $\{(1, 1, …, 1)\}$ is an orbit of size $1$. Hence there is an orbit $\mathcal{O} = \{(a_1, a_2, …, a_p)\}$ of size $1$ different than $\{(1, 1, …, 1)\}$, then $$(a_1, a_2, …, a_p) = \sigma(a_1, a_2, …, a_p) = (a_2, a_3, …, a_p, a_1)$$ so $a_1 = a_2 = … = a_p$. We have $a_1^p = 1$, so $a_1$ is an element of order $p$ in $G$.

Theorem 2. Let $G$ be a finite abelian group of order $n$. Then for every positive divisor $d$ of $n$, there is a subgroup of $G$ having order $d$.

Proof. We shall prove this theorem via induction on $n$.

Base case $n = 1$. There is nothing to prove.

Induction step. Assume that the theorem holds for every $n \leqslant n_0$ for some $n_0\in \mathbb{Z}^{+}$. Suppose that $G$ is a group of order $n_0 + 1$. Given $d$ a positive divisor of $n_0 + 1$. If $d = 1$, then there is nothing to prove, so suppose that $d > 1$. Then $d$ has a prime divisor $p$.

Since $d$ is a divisor of $n_0 + 1$, we have $p$ is a prime divisor of $n_0 + 1$. By Cauchy’s theorem, $G$ has an element $x$ of order $p$. Denote $\left<x\right>$ by $N$. Since $G$ is abelian, $N$ is a normal subgroup of $G$. The quotient group $G / N$ is also abelian, and has order $(n_0 + 1) / p < n_0$. By induction hypothesis, $G / N$ has a subgroup $H / N$ of order $d / p$. We have $$|H| = |N|\cdot [G : N] = p \cdot \dfrac{d}{p} = d$$ so $H$ is a subgroup of order $d$ in $G$.

Definition 1. A group $G$ is said to be simple if $|G| > 1$ and the only normal subgroups of $G$ are $1$ and $G$.

Proposition 1. If $G$ is an abelian simple group, then $G\cong \mathbb{Z}_p$ for some prime $p$.

Proposition 2. If $G$ is a solvable simple group, then $G\cong \mathbb{Z}_p$ for some prime $p$.

Definition 2. Let $G$ be a group. A composition series of $G$ $$1 = G_0 \leqslant G_1 \leqslant … \leqslant G_n = G$$ is a finite chain of subgroups of $G$ satisfying the following conditions

  1. $G_{i} \trianglelefteq G_{i + 1}$ for every $i\in \{0, 1, …, n - 1\}$.
  2. $G_{i + 1} / G_{i}$ is simple for every $i\in \{0, 1, …, n - 1\}$.

The quotient groups $G_{i + 1} / G_{i}$ are called composition factors of $G$.

If a chain of subgroups of $G$ satisfies only the first condition, then we say this chain a subnormal series, and write $$1 = G_0 \trianglelefteq G_1 \trianglelefteq … \trianglelefteq G_n = G$$

Definition 3. A group $G$ is said to be solvable if there is a subnormal series $$1 = G_0 \trianglelefteq G_1 \trianglelefteq … \trianglelefteq G_n = G$$ such that $G_{i + 1} / G_{i}$ are abelian.

Proposition 3. Let $G$ be a group, let $N$ be a normal subgroup of $G$. Then $G$ is solvable if and only if $N$ and $G / N$ are solvable.

Proof. Suppose that $G$ is solvable: $$1 = G_0 \trianglelefteq G_1 \trianglelefteq … \trianglelefteq G_n = G$$ with $G_{i + 1} / G_{i}$ are abelian.

For each $i\in \{0, 1, …, n - 1\}$, define $N_i = G_i \cap N$, then $$1 = N_0 \trianglelefteq N_1 \trianglelefteq … \trianglelefteq N_n = N$$ is a subnormal series that $N_{i +1} / N_{i}$ are abelian.

For each $i\in \{0, 1, …, n - 1\}$, define $M_i = G_iN$. Since $N$ is normal in $G$, we have $M_i$ are subgroups of $G$. Given $i\in \{0, 1, …, n - 1\}$, given $g\in G_{i + 1}$, we have $$gM_ig^{-1} = g(G_iN)g^{-1} = (gG_ig^{-1})(gNg^{-1}) = G_iN = M_i$$ so $G_{i + 1} \leqslant N_G(M_i)$. In another hand, $N \leqslant M_i \leqslant N_G(M_i)$, so $$M_{i + 1} = G_{i + 1}N = \left<G_{i + 1}, N\right> \leqslant N_G(M_i)$$ Hence $M_{i} \trianglelefteq M_{i + 1}$. Therefore $$N / N = M_0 / N \trianglelefteq M_1 / N \trianglelefteq … \trianglelefteq M_n / N = G / N$$ is a subnormal series. Given $i\in \{0, 1, …, n - 1\}$, by the third isomorphism theorem $$(M_{i + 1} / N) / (M_i / N)\cong M_{i + 1} / M_i$$

Consider the map $$ \begin{alignat*}{2} \varphi: G_{i + 1} & \to M_{i + 1} / M_i \\ g & \mapsto gM_i \end{alignat*} $$

This map is obviously a homomorphism. Given an element $gnM_i\in M_{i + 1} / M_i$ for some $g\in G_{i + 1}$ and $n\in N$, we have $M_i = G_iN = NG_i$, so $$gnM_i = gn(NG_i) = g(NG_i) = gM_i = \varphi(g)$$ Therefore $\varphi$ is surjective. Also, given $g\in G_{i}$, we have $$\varphi(g) = gM_i = g(G_iN) = G_iN$$ Hence $G_i \leqslant \ker\varphi$. Thus, $\varphi$ induces a homomorphism $\overline{\varphi}$ $$ \begin{alignat*}{2} \overline{\varphi} : G_{i + 1} / G_{i} & \to M_{i + 1} / M_i \\ gG_{i} & \mapsto \varphi(g) \end{alignat*} $$ that is surjective. Since $G_{i + 1} / G_i$ is abelian, we have $M_{i + 1} / M_i = \varphi(G_{i + 1} / G_i)$ is also abelian.

Therefore, $G / N$ is solvable. The other direction of the proposition is easy to solve.

Theorem 3. Let $G$ be a finite group and let $H$ be a normal subgroup of $G$. Then there is a composition series of $G$, one of whose terms is $H$.

Theorem 4. Let $G$ be a finite group. Then the following statements are equivalent

  1. $G$ is solvable.
  2. $G$ has a subnormal series $1 = G_0 \trianglelefteq G_1 \trianglelefteq … \trianglelefteq G_n = G$ such that $G_{i + 1} / G_{i}$ are cyclic.
  3. All composition factors of $G$ are of prime order.
  4. $G$ has a chain of subgroups $1 = G_0 \leqslant G_1 \leqslant … \leqslant G_n = G$ such that $G_i$ are normal in $G$ and $G_{i + 1} / G_{i}$ are abelian.

Proof. We show that $(1) \implies (3) \implies (2) \implies (1)$ and $(1)\iff (4)$ by induction on $|G|$.

We observe that $(3) \implies (2) \implies (1)$ and $(4)\implies (1)$ are trivial. So we only need to prove $(1)\implies (3)$ and $(1)\implies (4)$.

Base case $|G| = 1$. There is nothing to prove.

Induction step. Assume that the theorem holds for every group of order less than $n$. Let $G$ be a group of order $n$.

Prove $(1) \implies (3)$. If $G$ is simple, then there’s nothing to prove, so suppose that $G$ is not simple. Let $1 < H < G$ be a normal subgroup of $G$ and let $$1 = G_0 \trianglelefteq G_1 \trianglelefteq … \trianglelefteq G_n = G$$ be a composition series of $G$ where $G_k = H$ for some $k\in \{1, 2, …, n - 1\}$. Then $$1 = G_0 \trianglelefteq G_1 \trianglelefteq … \trianglelefteq G_k = H$$ is a composition series of $H$ and $$H / H = G_k / H \trianglelefteq G_{k + 1} / G \trianglelefteq … \trianglelefteq G_n / H = G / H$$ is a composition series of $G / H$. By the induction hypothesis, $G_{i + 1} / G_{i}$ are of prime order.

Prove $(1)\implies (4)$. Let $M$ be a non-trivial minimal normal subgroup of $G$: For every subgroup $N$ of $G$ such that $1 < N < M$, we have $N$ is not normal in $G$.

By $(3)$, since $M$ is solvable, $M$ as a normal subgroup $N$ of index $p$ for some prime $p$. Since $M$ is normal in $G$, for every $g\in G$, $gNg^{-1}$ is also a subgroup of index $p$ of $M$.

Given $x, y\in M$, we have $xyx^{-1}y^{-1}\in gNg^{-1}$ for every $g\in G$, so $$xyx^{-1}y^{-1}\in \bigcap\limits_{g\in G}gNg^{-1}$$

The subgroup $\bigcap\limits_{g\in G}gNg^{-1}$ is the largest normal subgroup of $G$ that contained in $N$. By the minimality of $G$, $\bigcap\limits_{g\in G}gNg^{-1} = 1$ so $xyx^{-1}y^{-1} = 1$. Hence, $M$ is abelian.

By induction hypothesis, $G / M$ has a chain of subgroups $$M / M = G_1 / M \leqslant G_2 / M \leqslant … \leqslant G_n / M = G / M$$ such that $G_i / M$ are normal in $G / M$ and $(G_{i + 1} / M) / (G_i / M)$ are abelian.

By the third isomorphism theorem and $M$ is abelian, $G_i$ are normal in $G$ and $G_{i + 1} / G_i$ are abelian.

Theorem 5. Let $G$ be a finite group. Assume that $G$ has two composition series $$1 = N_0 \trianglelefteq N_1 \trianglelefteq … \trianglelefteq N_r = G$$ and $$1 = M_0 \trianglelefteq M_1 \trianglelefteq … \trianglelefteq M_s = G$$ then $r = s$ and there exists a permutation $\sigma$ on $\{0, 1, …, r\}$ such that $$N_{i + 1} / N_i \cong M_{\sigma(i) + 1} / M_{\sigma(i)}$$ for every $i\in \{0, 1, …, r - 1\}$.

Proof. For short, we abbreviate the condition in the theorem to that two given composition series are equivalent. It’s easy to see that, if a composition series $\alpha$ is equivalent to a composition series $\beta$, and $\beta$ is equivalent to a composition series $\gamma$, then $\alpha$ is equivalent to $\gamma$.

We shall prove this theorem by induction on $\min\{r, s\}$.

Without loss of general, assume that $s \leqslant r$.

Base case $s = 1$. There is nothing to prove.

Induction step. Assume that the theorem is true for every $s < n$ for some $n$. Consider $s = n$.

We have $N_{r - 1}M_{s - 1}$ is a normal subgroup of $G$ that contains $M_{s - 1}$, so $N_{r - 1}M_{s - 1} = M_{s - 1}$ or $N_{r - 1}M_{s - 1} = G$.

In the case $N_{r - 1}M_{s - 1} = M_{s - 1}$, we have $N_{r - 1} \leqslant M_{s - 1}$. We have $M_{s - 1}$ is a normal subgroup of $G$ that contains $N_{r - 1}$, so $M_{s - 1} = N_{r - 1}$. By induction hypothesis, the theorem holds in this case.

In the case $N_{r - 1}M_{s - 1} = G$, we have $$N_{r - 1} / (N_{r - 1} \cap M_{s - 1}) \cong N_{r - 1}M_{s - 1} / M_{s - 1} = G / M_{s - 1}$$ and $$M_{s - 1} / (N_{r - 1} \cap M_{s - 1}) \cong M_{s - 1}N_{r - 1} / N_{r - 1} \cong G / N_{r - 1}$$

Let $H = N_{r - 1} \cap M_{s - 1}$, let $$1 = H_0 \trianglelefteq H_1 \trianglelefteq … \trianglelefteq H_k = H$$ be a composition series of $H$. Then $$1 = H_0 \trianglelefteq H_1 \trianglelefteq … \trianglelefteq H_k = H \trianglelefteq N_{r - 1}$$ is a composition series of $N_{r - 1}$ and $$1 = H_0 \trianglelefteq H_1 \trianglelefteq … \trianglelefteq H_k = H \trianglelefteq M_{s - 1}$$ is a composition series of $M_{s - 1}$.

By induction hypothesis on $N_{r - 1}$, the composition series $$1 = H_0 \trianglelefteq H_1 \trianglelefteq … \trianglelefteq H_k = H \trianglelefteq N_{r - 1}$$ is equivalent to $$1 = N_0 \trianglelefteq N_1 \trianglelefteq … \trianglelefteq N_{r - 1}$$ Hence two composition series $$1 = H_0 \trianglelefteq H_1 \trianglelefteq … \trianglelefteq H_k = H \trianglelefteq N_{r - 1} \trianglelefteq G$$ and $$1 = N_0 \trianglelefteq N_1 \trianglelefteq … \trianglelefteq N_{r} = G$$ are equivalent.

Similarly, two composition series $$1 = H_0 \trianglelefteq H_1 \trianglelefteq … \trianglelefteq H_k = H \trianglelefteq M_{s - 1} \trianglelefteq G$$ and $$1 = M_0 \trianglelefteq M_1 \trianglelefteq … \trianglelefteq M_{s} = G$$ are equivalent.

In another hand, two composition series $$1 = H_0 \trianglelefteq H_1 \trianglelefteq … \trianglelefteq H_k = H \trianglelefteq N_{r - 1} \trianglelefteq G$$ and $$1 = H_0 \trianglelefteq H_1 \trianglelefteq … \trianglelefteq H_k = H \trianglelefteq M_{s - 1} \trianglelefteq G$$ are equivalent.

Therefore two composition series $$1 = N_0 \trianglelefteq N_1 \trianglelefteq … \trianglelefteq N_r = G$$ and $$1 = M_0 \trianglelefteq M_1 \trianglelefteq … \trianglelefteq M_s = G$$ are equivalent.
