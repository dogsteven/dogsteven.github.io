Group Actions 🔗

Definition 1. Let $A$ be a set. The set of all bijections from $A$ into itself, denoted $S_A$, is a group under map composition, called the symmetric group of $A$. Each element of $S_A$ is called a permutation on $A$.

Definition 2. An action of a group $G$ on a set $A$ is a map $$ \begin{alignat*}{2} \cdot : G\times A & \to A \\ (g, a) & \mapsto g\cdot a \end{alignat*} $$ satisfying the following properties

  1. $g_1\cdot (g_2\cdot a) = (g_1g_2)\cdot a$ for every $g_1, g_2\in G, a\in A$.
  2. $1\cdot a = a$ for every $a\in A$.

Informally, we say that $G$ acts on $A$.

Proposition 1. Assume that a group $G$ acts on a set $A$. Then for every $g\in G$, the map $$ \begin{alignat*}{2} \sigma_g : A & \to A \\ a & \mapsto g\cdot a \end{alignat*} $$ is a permutation on $A$. Furthermore, for every $g_1, g_2\in G$, $$\sigma_{g_1g_2} = \sigma_{g_1}\circ \sigma_{g_2}$$ Thus, the map $$ \begin{alignat*}{2} \sigma: G & \to S_A \\ \sigma & \mapsto \sigma_g \end{alignat*} $$ is a homomorphism from $G$ into the symmetric group of $A$, which is called the permutation representation associated with the given action.

Definition 3. Let $G$ acts on a set $A$. Define a relation $\sim$ on $A$ by $$a \sim b \iff (\exists g\in G)a = g\cdot b$$ It’s east to see that this relation in an equivalence relation whose equivalence classes called orbits of the given action. The set of all orbits is denoted by $A / G$. For an elemnet $a\in A$, the orbit containing $a$ is denoted by $Ga$: $$Ga \overset{\text{def}}{=} \left\{g\cdot a : g\in G\right\}$$

Given $a\in A$, the stablizer of $a$, denoted $G_a$, is defined to be the set $$G_a = \left\{g\in A : g \cdot a = a\right\}$$ It’s easy to see that stablizers are subgroups of $G$.

The kernel of the given action is defined to be the kernel of the corresponding permutation representation, which equals to $$\bigcap\limits_{a\in A} G_a$$

An action is said to be faithful if its kernel is identity. An action is said to be transitive if it has only one orbit.

Proposition 2. A group $G$ acts on itself by conjugation: $$g\cdot a = gag^{-1}$$ The stablizer of an element $a\in G$ is the centralizer of $a$ in $G$. The orbits of this action are called conjugacy classes of $G$. The kernel of this actions is called the center of $G$, denoted by $Z(G)$. $$Z(G) = \left\{g\in G : ga = ag \text{ for every }a\in G\right\}$$ One can see that $Z(G)$ is a normal subgroup $G$ that is abelian.

A conjugacy class either contains only one element which is an centre element, or consists of more than one non-centre elements.

Proposition 3. A group $G$ acts on $\mathcal{P}(G) \setminus \{\varnothing\}$ by conjugation: $$g\cdot A = gAg^{-1}$$ The stablizer of a non-empty subset $A$ of $G$ is the normalizer of $A$ in $G$.

Theorem 1. Let a group $G$ act on a set $A$, let $a$ be an element of $A$. Then for every element $a\in A$, we have $$|Ga| = [G : G_a]$$

Proof. Given $a\in A$, consider the map $$ \begin{alignat*}{2} \varphi: G / G_a & \to Ga \\ gG_a & \mapsto g\cdot a \end{alignat*} $$

Given $g_1, g_2\in G$ such that $g_1G_a = g_2G_a$, then $g_2^{-1}g_1\in G_a$, so $$\varphi(g_1G_a) = g_1\cdot a = g_2\cdot \left(\left(g_2^{-1}g_1\right)\cdot a\right) = g_2\cdot a = \varphi(g_2G_a)$$ Hence $\varphi$ is injective.

Given $b\in Ga$, there is an element $g\in G$ such that $b = g\cdot a$, then $$\varphi(gG_a) = g\cdot a = b$$ so $\varphi$ is surjective.

Therefore, $\varphi$ is bijective, which implies that $$|Ga| = |G / G_a| = [G : G_a]$$

Theorem 2. Let $G$ be a finite group, let $g_1, g_2, …, g_r$ be representatives of non-centre conjugacy classes. Then $$|G| = |Z(G)| + \sum\limits_{i = 1}^{r} [G : C_G(g_i)]$$
